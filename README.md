# Parallel Guardian AI

An AI-powered academic productivity assistant that helps students avoid missed deadlines through intelligent planning, risk prediction, rescue workflows, and daily reflections.

## Features

- Smart task capture using natural language
- AI-generated daily brief
- Deadline Guardian risk detection
- AI Priority Queue
- Emergency Rescue plans
- Daily Reflection Agent
- Productivity analytics
- Explainable AI confidence scores
- Personalized focus windows

## Tech Stack

- React + TypeScript
- Vite
- Supabase
- Google Gemini API
- Lovable
